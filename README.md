# 🏢 Sentiric Agent Service

Konuşma Uzmanı. Workflow tarafından tetiklendiğinde İnsan-Makine Etkileşimi (Handover) anını yönetir.

## 🚀 Hızlı Başlangıç
```bash
go run cmd/agent-service/main.go
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/services/agent.spec.yaml](https://github.com/sentiric/sentiric-spec)
